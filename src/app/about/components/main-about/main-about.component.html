Laman web ini ialah laman web Bahasa Malaysia untuk semakan hadis. Tim semakhadis.com berusaha menghimpunkan semakan-semakan hadis dari sumber yang dipercayai bagi rujukan umat Islam. Setiap hadis yang ada di website ini disertakan dengan sumber rujukan bagi membolehkan semakan dilakukan. Orang awam boleh membuat pertanyaan dan pembetulan menggunakan borang maklumbalas yang kami sediakan.

Sumber kami adalah daripada rujukan seperti berikut :

Website Hadis

Laman-laman Web Semakan Hadis Bahasa Arab seperti Durar Al-Saniyyah, Al-Mawsu’ah Al-Hadithiyyah dan Hadith.

Buku-Buku Hadis

Laman-laman Web Semakan Hadis Bahasa Arab seperti Durar Al-Saniyyah, Al-Mawsu’ah Al-Hadithiyyah dan Hadith.

Kajian

Kajian-kajian individu dari kalangan penuntut ilmu yang mendalami bidang hadis.

Tesis

Tesis kajian pelajar universiti dalam bidang hadis.
